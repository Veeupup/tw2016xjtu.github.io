vulhub搭建

Vulhub是一个基于docker和docker-compose的漏洞环境集合，进入对应目录并执行一条语句即可启动一个全新的漏洞环境，让漏洞复现变得更加简单，让安全研究者更加专注于漏洞原理本身。

  • 虚拟机环境 VirtualBox : ubuntu 18.04LTS
  • 运行环境：docker
  • 工具
    • docker-compose 利用开源靶场image
    • docker
  • 靶场环境：由 vulhub 提供
